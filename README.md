# BI - DataObject

## Setting up dev

### Clone repository

```
git clone git@github.com:CPNV-ES-BI/BI_PYTHON_AZURE.git
cd BI_PYTHON_AZURE/
```

### Run Unittests

#### Locally

```shell
python -m unittest discover tests/
```
